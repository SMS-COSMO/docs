# API

- `POST /check`
- `POST /add`

`check` 为检查论文相似度

`add` 为 `check` 之后将论文加入数据库 **!!暂不支持修改!!**

**请求:**

```json
{
    "id": "1",
    "text": "基于 jieba 分词，tf-idf 算法求相似度   ！（*@1=-+!9）"
}
```

其中 id 应不与已有论文重复。

**响应:**

```json
{
    "similarity": [
        [
            0.97821377847197237,
            "3"
        ],
        [
            0.23219838816700674,
            "1"
        ],
        [
            0.09556481196700674,
            "5"
        ],
        [
            0.0009258006340497831,
            "9"
        ],
        [
            0,
            "4"
        ]
    ]
}
```

表示前 5 个相似度最高的论文，每项第一个值表示相似度，第二个值为论文 id。
